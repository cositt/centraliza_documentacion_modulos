# account_statement_import_base

Base para importar archivos de extractos.

## ¿Qué es?

Wizard y utilidades para importación.

## Instalación

```
Aplicaciones → Buscar "Statement Import"
Instalar: account_statement_import_base
```

## Uso

```
Contabilidad → Extractos Bancarios
→ Importar
```
