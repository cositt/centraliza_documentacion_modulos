# account_statement_import_file

Importar archivos PDF/TXT de extractos.

## ¿Qué es?

Carga de PDFs y textos:

- OCR opcional
- Parsing texto
- Importar movimientos
- Crear extracto

## Instalación

```
Aplicaciones → Buscar "Import Statement Files"
Instalar: account_statement_import_file
```

## Uso

```
Contabilidad → Extractos → Importar
→ Subir archivo PDF/TXT
→ Mapear campos
→ Crear movimientos
```
