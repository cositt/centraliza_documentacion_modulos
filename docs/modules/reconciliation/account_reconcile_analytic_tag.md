# account_reconcile_analytic_tag

Tags analíticos durante reconciliación.

## ¿Qué es?

Asignar etiquetas analíticas:

- Por centro de costos
- Por proyecto
- Por departamento
- En reconciliación

## Instalación

```
Aplicaciones → Buscar "Reconcile Analytic"
Instalar: account_reconcile_analytic_tag
```

## Uso

Durante reconciliación:

```
Factura: 500€ (Proveedor materiales)
├─ Etiqueta: Producción
├─ Etiqueta: Planta Madrid
└─ Etiqueta: Q2-2026
```
